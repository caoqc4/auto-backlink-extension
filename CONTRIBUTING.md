# Contributing

Thanks for taking a look at Backlink Forge.

## Development

```bash
npm install
npm run build
```

Please keep changes scoped and include a short note about the workflow you tested. For extension behavior, mention whether you tested the popup, side panel, content script, Google Sheets sync, or page screening flow.

## Pull Requests

- Avoid committing personal backlink data, browser exports, cookies, API keys, OAuth tokens, or screenshots containing private account information.
- Keep generated build output out of the repository.
- Update the README when a change affects setup, permissions, storage, or user-visible workflows.
- Prefer small PRs that are easy to review.

## Security

If you find a security issue or a privacy leak, please do not open a public issue with exploit details. Contact the maintainer privately first.
